## 🌟 What is this?

> **GitHub Portfolio Generator** is a **zero-backend**, browser-only tool that lets developers instantly transform any GitHub repository URL into a fully structured, export-ready project listing.

No server. No login. No setup. Just open the HTML file and go.

Whether you're a **developer building your portfolio**, a **recruiter reviewing projects**, or a **student organizing their work** — this tool saves you hours.

## ✨ Features

<table>
<tr>
<td width="50%">

### ⚡ Core

## 🚀 Getting Started

### Prerequisites

```
✅ Any modern browser  (Chrome 90+, Firefox 88+, Edge 90+, Safari 14+)
✅ Internet connection (for GitHub API calls)
❌ No Node.js needed
❌ No npm needed
❌ No backend needed
```

### Installation

```bash
# Clone the repository
git clone https://github.com/deepakrajjs-29/github-portfolio-generator.git

# Navigate into it
cd github-portfolio-generator

# Open in browser — that's it!
open index.html          # macOS
start index.html         # Windows
xdg-open index.html      # Linux
```

Or simply **[Download ZIP](https://github.com/deepakrajjs-29/github-portfolio-generator/archive/refs/heads/main.zip)** and open `index.html`.

## 💻 How to Use

### 1️⃣ Single URL Mode

```
Step 1 → Paste:   https://github.com/username/repository-name
Step 2 → Click:   "Extract & Add"
Step 3 → Done:    Stars, forks, topics, live link — all auto-filled ✨
```

### 2️⃣ Bulk Paste Mode

```
Paste multiple URLs (one per line):

  https://github.com/user/project-alpha
  https://github.com/user/project-beta
  https://github.com/another/awesome-repo
  https://github.com/someone/cool-tool

Click "Extract All" → fetches all in parallel with live progress bar!
```

### 3️⃣ Export Your Portfolio

```
📗 CSV    →  Open in Excel / Google Sheets for further editing
📕 PDF    →  Share as a professional document
📘 Word   →  Customise in Microsoft Word / Google Docs
📙 JSON   →  Feed into your own website or API
```
